# SkyFuel

SkyFuel est l'application indispensable pour les propriétaires de drones qui veulent s'assurer que leurs batteries sont toujours prêtes à prendre leur envol ! Avec SkyFuel, vous pouvez gérer toutes vos batteries de drone en un clin d'œil depuis votre téléphone.

## Fonctionnalités clés

- Enregistrement de toutes les infos utiles pour chaque batterie : marque, niveau de charge (full, low ou storage), nombre de cycles et même leur petit nom si vous voulez !
- Suivi en temps réel de l'état de charge de chaque batterie, pour ne jamais être pris au dépourvu. 
- Notification pour éviter les accidents : pas question de laisser votre batterie se décharger ou se charger trop longtemps !
- Organisation facile de toutes les batteries de drone en un seul endroit, pour ne plus jamais les perdre.

## Configuration requise

Rien de bien compliqué, juste un téléphone qui fonctionne et un brin de bonne humeur !

## Installation

Aucune installation possible pour le moment, mais vous pouvez toujours nous contacter pour nous faire part de vos suggestions !

## Utilisation

1. Enregistrez toutes vos batteries de drone dans l'application.
2. Ajoutez toutes les infos utiles pour chaque batterie : marque, niveau de charge actuel, nombre de cycles et tout ce qui vous semble pertinent. 
3. Coller les étiquettes sur les batteries.
4. Scannez les étiquettes avec l'application pour voir les infos de la batterie scannée.
5. Recevez des notifications pour ne jamais laisser une batterie décharger ou charger trop longtemps. 
6. Organisez facilement toutes vos batteries de drone en un seul endroit à partir de l'application, pour être toujours prêt à prendre votre envol !
